# Astrea - Sothis


---

A small example project culminating all the things I've learned are good, bad, fun, and unfun in Slay the Spire into a small "early" character with roughly 30 functioning cards to provide a 
cohesive gameplay experience with good player feedback and multiple options.

The goal was to create interesting, coherent, and unique gameplay mechanics that fit the core Slay the Spire mold while minimizing extra UI elements to take up as little space as possible and provide information as quickly as possible.
